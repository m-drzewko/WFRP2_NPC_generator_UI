Welcome to

Warhammer FRP 2ED NPC Generator App

This application will enable you to quickly generate a random character for your WFRP session. Whether you need a courageous human, a stout dwarf, a reclusive elf or jovial halfling, this application will generate a stat block alongside the basic visual characteristics of a character.

Roadmap for the future:
- create a name generation component
- add login functionality
- add NPC save feature
- add character archetypes to generation parameters
